# 01. Паспорт проекта (Project Overview)

## Введение
Веб-платформа для приюта бездомных животных "ПИФ" (Донецк).
**Миссия:** Цифровизация процессов приюта для увеличения числа пристройств и обеспечения финансовой прозрачности. Платформа служит мостом между животными и потенциальными хозяевами/донорами.

## Ключевые метрики (KPI)
1.  **Пристройство:** Выход на показатель 150 животных в год.
2.  **Финансы:** Сбор не менее 1 000 000 ₽ пожертвований ежегодно.
3.  **Опекунство:** Увеличение базы постоянных опекунов (рекуррентные платежи).

## Целевая аудитория
1.  **Потенциальные хозяева:** Люди, ищущие питомца. Им важен удобный поиск, качественные фото и честное описание характера.
2.  **Доноры:** Люди, готовые помочь деньгами, но требующие прозрачности (куда пошли деньги).
3.  **Волонтеры:** Сотрудники, которым нужен удобный инструмент для ведения базы (вместо бумажных журналов или Excel), работающий с телефона.

## Архитектура решения (High-Level)
Система состоит из трех компонентов (Monorepo):
1.  **Backend API (NestJS):** Центральный мозг системы. REST API, работа с БД, интеграция платежей и AI.
2.  **Frontend (Клиентский сайт):** Публичная часть для гостей и опекунов. Фокус на UX, SEO и эмоциях.
3.  **Admin Panel (PWA):** Мобильное веб-приложение для сотрудников. Фокус на быстродействии и работе "на ходу". Реализовано на Refine.

## Основной функционал (Scope)

### MVP (Этап 1)
* Каталог животных с фильтрами.
* Профили животных (фото, история, нужды).
* Система донатов (эквайринг, подписки).
* Админ-панель для управления базой.
* Telegram-бот для отчетов опекунам.
* Базовый AI-поиск (векторный поиск по описанию).
* Черный список (по номеру телефона).
* Анкетирование ("Fast Track" заявки).
