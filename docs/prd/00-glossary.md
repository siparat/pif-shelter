# 00. Глоссарий проекта "Приют ПИФ"

Этот документ фиксирует единую терминологию для разработчиков, дизайнеров и заказчика, чтобы избежать разночтений.

## Сущности и Акторы

### Роли пользователей
* **Гость (Guest):** Неавторизованный посетитель сайта. Может искать животных, донатить, оставлять заявки.
* **Опекун (Guardian):** Пользователь, оформивший регулярную финансовую подписку на конкретное животное. Получает отчеты через Telegram-бота.
* **Волонтер (Volunteer):** Сотрудник приюта с базовыми правами доступа в админ-панель. Отвечает за обновление статусов и контент подопечных.
* **Старший волонтер (Senior Volunteer):** Опытный сотрудник. Может создавать сборы, редактировать любые карточки животных, модерировать контент.
* **Администратор (Admin):** Технический и операционный руководитель платформы. Имеет полные права, включая управление доступами (создание аккаунтов персонала).

### Животные
* **Подопечный:** Животное, находящееся на балансе приюта.
* **Карточка животного:** Страница с фото, описанием, статусом и историей.
* **Выпускник:** Животное, которое успешно нашло дом (статус "Пристроен").
* **На радуге:** Животное, которое погибло/умерло (архивный статус).

### Финансы и Помощь
* **Донат (Donation):** Добровольное пожертвование (разовое или регулярное).
* **Целевой сбор (Fundraising):** Сбор средств с конкретной целью, суммой и дедлайном (напр., "На операцию Шарику").
* **Опекунство (Guardianship):** Ежемесячная подписка на фикс. сумму (3200₽), покрывающая базовые нужды конкретного животного.
* **Помощь вещами (In-kind donation):** Передача материальных ценностей (корм, лекарства) согласно списку "Всегда нужно".

## Процессы
* **Пристройство:** Процесс передачи животного новому владельцу (Заявка -> Собеседование -> Договор -> Передача).
* **Fast Track (Анкета):** Заполнение расширенной анкеты пользователем для повышения приоритета его заявки на знакомство.
* **Возврат:** Ситуация, когда животное возвращают в приют. Может повлечь занесение владельца в Черный список.
